PCAP-observation
-----------------
- Om du laddade ner `lab2-modbus.pcap` men inte hittar några Modbus-paket (FC3/FC6/FC16), dokumentera detta här. Vid detta tillfälle innehöll den nedladdade pcap:en inga Modbus-transaktioner, så ingen `wireshark-fc3.png` skapades.
- Tips: försök hämta pcap:en igen senare under workshopdagen eller använd live-webbgränssnittet för att ta HMI-/alerts-skärmdumpar i stället.

Om du vill kan du också lägga till en skiss av zonindelningen i `screenshots/zone-sketch.jpg`.
